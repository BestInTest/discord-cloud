# Discord Cloud - store and share files

# How to use
1. You need to have Java 17+ installed.
2. Download the latest release from [here](https://github.com/BestInTest/discord-cloud/releases).
3. Extract downloaded archive.
4. Run `discord-cloud-gui`
   <br>

You can also use modules in cli. Just run them using `java -jar <file>` command:
```
java -jar uploader-2.0.jar
```

# Compiling
Run `mvn clean package` in main project directory.
<br>
Compiled files will be in `build` directory.
